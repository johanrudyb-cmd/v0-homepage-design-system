# Stack technique - OUTFITY

- **Framework** : Next.js 16 (App Router, Turbopack)
- **Langage** : TypeScript
- **UI** : Tailwind CSS, shadcn/ui (Radix), Lucide icons
- **Base de données** : PostgreSQL (Supabase) + Prisma
- **Auth** : NextAuth (sessions, plans)
- **IA** : OpenAI (ChatGPT) pour analyses, scripts, tech packs
- **Déploiement** : Vercel (ou équivalent)

Détails complets : `docs/saas-architecture.md`.
